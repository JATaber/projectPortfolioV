About

Welcome to my project for Project and Portfolio V, titled Stock Image Finder. The purpose of this site is to show some of the skills that I have learned up to this point. This page utilizes an API call to 3 different sites to return results for the image that the user is searching for. Responsive Web Design has been implemented so that this site can be viewed on multiple screen sizes and responsive forms to give the user feedback if they have left a field blank.

Search

The search feature searches 3 APIs to return stock images per what you want to search for. The three APIs are for UnSplash, Pixabay, and Flickr

Contact

The Contact page is to demostrate a responsive form and doesn't send an actual email to anyone. It will also display a popup of sorts when the form is completed.
